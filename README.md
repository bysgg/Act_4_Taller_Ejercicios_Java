# 🧠 Proyectos de Programación Orientada a Objetos (POO)

Este repositorio contiene dos ejercicios desarrollados en Java como parte del aprendizaje de Programación Orientada a Objetos. Cada proyecto simula un sistema real aplicando conceptos como encapsulamiento, herencia, polimorfismo y abstracción.

---

## 📂 Contenido del Repositorio

```
Ejercicios_POO/
├── Ejercicio_1_Televentas/
└── Ejercicio_2_Museo/
```

---

## 📦 Ejercicio 1: Sistema de Televentas

Simulación de un sistema de ventas que permite gestionar productos, órdenes de compra, pagos, logística y quejas de clientes.

### 🔹 Funcionalidades

* Gestión de catálogo de productos
* Creación de órdenes de compra
* Procesamiento de pagos (Strategy)
* Control de estados de la orden
* Simulación de logística
* Registro de quejas

### 🧩 Conceptos Aplicados

* Interfaces (`MetodoPago`)
* Enumeraciones (`EstadoOrden`)
* Separación de responsabilidades
* Patrón Strategy (métodos de pago)

---

## 🏛️ Ejercicio 2: Sistema de Gestión de Museo

Simulación de un sistema para administrar obras de arte, usuarios y procesos de restauración dentro de un museo.

### 🔹 Funcionalidades

* Gestión de obras (cuadros y esculturas)
* Control de usuarios con roles
* Registro de restauraciones
* Historial de mantenimiento
* Valoración total del inventario
* Cesión de obras

### 🧩 Conceptos Aplicados

* Herencia (`ObraDeArte`, `Cuadro`, `Escultura`)
* Polimorfismo
* Clases abstractas
* Manejo de listas y composición

---

## 🛠️ Tecnologías Utilizadas

* Java 8+
* Programación Orientada a Objetos (POO)

---

## ▶️ Cómo Ejecutar

1. Clonar el repositorio
2. Compilar los archivos:

```
javac */com/*/*.java
```

3. Ejecutar cada proyecto:

### Televentas

```
java Ejercicio_1_Televentas.com.televentas.Main
```

### Museo

```
java Ejercicio_2_Museo.com.museo.Main
```

## 📌 Objetivo Académico

Estos ejercicios fueron desarrollados con el fin de:

* Comprender los principios de la POO
* Modelar problemas del mundo real en código
* Aplicar buenas prácticas de diseño
* Prepararse para proyectos más complejos
